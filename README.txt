PETA - Preschool Educational Training Academy

Requirements
Must have java sdk installed
Must have msyql installed

Program Description
This is a mini project on an enrollment system which makes use of database.
It involves the use of stored procedures and triggers in the database.
basic functionalities are create, update and delete of entries.
The GUI shows a list of entries.

Setting up the program
Import the pet_academy.sql in the database folder into your database.
copy the source code from the src folder into your workspace
add the database connector jar from the libraries folder into your build path
